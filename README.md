### gulp 使用指南
#### 项目启动
```js
// 1,安装node包
npm install
// 2,安装静态资源依赖包
bower install
// 3,启动代理服务
gulp serve
// 4,开发完成之后,编译生产代码
gulp
// 5,编译完成之后,启动生产环境服务,复验
gulp serve:dist
// 6,提交代码
Nice! Your work has been finished
```
#### 规范
由于现在还没有部署持续集成,所以每次提交代码时,需要提交一份编译完成的代码.避免环境带来的不便
