### ngSinaEmoji
整合新浪远程表情组件
####Useage
```js
 <textarea ng-model="emoji" ng-sina-emoji target="#emotion2" app-key="1362404091" cols="50" rows="5"></textarea>
<button id="emotion">表情</button>
```
#### Params
| key  |           value           |
| :--: | :--------------------: |
|  target  |          时间触发元素          |
|  appKey  |         新浪授权ID(可不填，有默认ID)          |
